# Nearby

Nearby é um aplicativo mobile desenvolvido com **React Native** e **Expo**, criado como parte do programa NLW Mobile da **Rocketseat**. O app tem como objetivo ajudar usuários a visualizar estabelecimentos parceiros próximos à sua localização e resgatar cupons de desconto diretamente pelo app.

---

## O que é o Nearby?

O **Nearby** é um aplicativo para visualização de estabelecimentos parceiros próximos e para resgatar cupons de desconto de maneira prática e intuitiva. É uma solução para facilitar o acesso a promoções e informações de estabelecimentos locais.

---

## Principais Funcionalidades

- **Visualizar Estabelecimentos Próximos:** Veja quais estabelecimentos parceiros estão perto de sua localização em tempo real.
- **Resgatar Cupons de Desconto:** Utilize os recursos da câmera para escanear QR Codes e resgatar cupons exclusivos.
- **Integração com APIs externas:** Dados são carregados dinamicamente através de conexão com APIs construídas em **Node.js**.
- **Listas roláveis & UI responsiva:** Uso de componentes como Bottom Sheet e roteamento dinâmico com o **Expo Router**.
- **Leitura de QR Code com a câmera:** Funcionalidade que utiliza a câmera para oferecer uma experiência amigável e direta ao usuário.
